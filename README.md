## ✨ Возможности
- 🎵 **Игра с клавиатуры** — три ряда русских букв соответствуют трём октавам пианино (от C3 до G5)
- 📖 **Нотная тетрадь** — 5 страниц с мелодиями, записанными буквами клавиатуры
- 🕯️ **Атмосфера** — анимированные свечи, викторианское оформление, плавные переходы
- 📃 **Звук перелистывания** — процедурно сгенерированный шум бумаги при смене страницы
- 🦶 **Педаль сустейна** — удерживайте `Пробел` для продления звучания нот
- 🖱️ **Мышь и тач** — кликайте по клавишам пианино или по буквам в тетради
- 🔊 **Регулировка громкости** — встроенный ползунок

## ⌨️ Управление

| Ряд клавиш | Диапазон нот | Клавиши |
|:-----------|:-------------|:--------|
| 🟤 Нижний  | C3 → B3      | `Я Ч С М И Т Ь Б Ю` |
| 🟡 Средний | C4 → B4      | `Ф Ы В А П Р О Л Д Ж Э` |
| 🔴 Верхний | C5 → G5      | `Й Ц У К Е Н Г Ш Щ З Х Ъ` |
| ⚪ Педаль  | Сустейн      | `Пробел` |

💡 *Нажмите на любую букву в нотной тетради, чтобы мгновенно услышать ноту.*

## 🚀 Запуск

Проект не требует сборки, зависимостей или бэкенда. Это один самодостаточный `index.html`.

### Локально
```bash
# 1. Скачайте или склонируйте репозиторий
git clone https://github.com/ваш-username/victorian-piano.git
cd victorian-piano

# 2. Откройте файл в браузере
open index.html      # macOS
start index.html     # Windows
xdg-open index.html  # Linux
